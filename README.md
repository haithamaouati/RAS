## Reverse

###### Function
Here is a function flow chart:

```mermaid
flowchart LR;
A[الهأ] -->|Reverse| B((أهلا));
A<-->|Vice Versa|B;
B-->|Reverse|A;
```

>**Note**:
> When entering the reversed Arabic text, it will be reversed and vice versa.
